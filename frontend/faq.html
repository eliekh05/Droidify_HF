Frequently Asked Questions

About Droidify, custom ROMs, rooting, and recoveries.

About Droidify

What is Droidify?

Droidify is a live index of custom ROMs, recoveries, root tools, and guides for Android devices. It fetches data from 20+ public sources in real time — LineageOS, OrangeFox, TWRP, crDroid, /e/OS, postmarketOS, GrapheneOS, and more. No data is hardcoded. No account required.

Who is it for?

Anyone who wants to flash a custom ROM, install a custom recovery, root their phone, or find stock firmware. Experienced flashers use it to quickly find what's available for a device. Beginners use it to discover what options exist before diving into XDA threads.

Is Droidify free?

Completely. No account, no subscription, no paywall, no ads. The source code is MIT-licensed on GitHub. You can self-host it with one command: ./install.sh after cloning.

Is Droidify affiliated with LineageOS, TWRP, or any ROM project?

No. Droidify is an independent project with no ties to any ROM team or Android organization. It reads their public APIs and websites the same way a browser would. Every trademark shown belongs to its respective owner.

Data & Sources

Where does the data come from?

Directly from the source projects — no middleman, no manual entry. Sources include: LineageOS Download API and Wiki, OrangeFox v3 API, TWRP search.json, crDroid website, SourceForge file listings for 30+ ROM projects, GrapheneOS, postmarketOS GitLab, DivestOS, CalyxOS, /e/OS, Ubuntu Touch, Kali NetHunter, customrombay.org, and more. Everything is fetched live and cached for up to two hours.

Why does my device show zero ROMs?

Several reasons are possible:

  • The ROM only exists as a forum post or Google Drive link — those can't be indexed automatically.
  • The cache hasn't populated yet — wait two minutes after a fresh server start.
  • The device genuinely has no active community ROM support.
  • The ROM uses a different codename than what Droidify has for your device.

If a ROM exists that should be showing, open a GitHub issue with the codename and the ROM's public download URL.

Why is the first page load slow?

On a cold start the backend fetches data from 20+ external APIs simultaneously. This warmup takes roughly two minutes. Once the cache is hot, every request responds in milliseconds. Subsequent restarts load from a saved cache file and are much faster.

How often is data updated?

Most sources refresh every two hours. Stock firmware links (SamFW) are cached for 24 hours since they change rarely. If a ROM build was released an hour ago it may not appear until the next cache cycle.

Can I trust the download links?

Droidify links to the official pages of each ROM project — TWRP's own server, OrangeFox's CDN, LineageOS mirrors. We don't proxy, mirror, or modify files. Always verify the SHA256 checksum against what the project publishes before flashing.

Custom ROMs & Rooting

What is a custom ROM?

A custom ROM is a modified version of Android compiled and distributed by a community developer instead of your device's manufacturer. Examples: LineageOS (clean AOSP-based), GrapheneOS (hardened for privacy), crDroid (feature-rich), /e/OS (Google-free). They replace the stock firmware entirely.

What is a custom recovery?

A custom recovery (TWRP, OrangeFox, PBRP) replaces the stock recovery partition. It lets you flash ZIP files, create full backups, wipe partitions, and install ROMs without booting into Android. It's usually the first thing you flash after unlocking the bootloader.

What does "rooting" mean?

Rooting gives you superuser (root) access to Android — equivalent to administrator access on a PC. With root you can modify system files, remove bloatware, use Tasker automations that need privileged access, and run apps that require root. Modern root tools: Magisk (most compatible), KernelSU (kernel-level), APatch (kernel patch).

Will flashing a custom ROM void my warranty?

On most devices, yes — unlocking the bootloader is a one-way hardware flag on Samsung (Knox 0x1) and some other manufacturers. On Google Pixels, the bootloader can be re-locked, which restores the device to a factory-equivalent security state. Check your device's specific situation before proceeding.

What does "bootloader unlock" mean?

The bootloader is the first thing that runs when your phone powers on. By default it only allows signed software from the manufacturer. Unlocking it lets you flash custom software. This wipes your device completely and on some devices (Verizon Samsung, for example) is permanently blocked by the carrier.

Android Versions

What is an Android API level?

Every Android release has an API level number. Android 14 = API 34, Android 15 = API 35, Android 16 = API 36. Droidify shows the Android version number, not the API level. When a source only provides an API level, it's converted automatically.

Why does a ROM show "Android unknown"?

Some ROM sources (customrombay.org, SourceForge builds, /e/OS community builds) don't publish which Android version a build targets in their structured data. Droidify shows "unknown" honestly rather than guessing. Check the ROM's download page for the actual version.

Contributing

Can I request a missing ROM or device?

Yes. Open an issue on GitHub with the device codename and the ROM project's public download URL or API endpoint. If the project has a structured file listing or JSON API, it can usually be added as a source. Single-file Google Drive links or forum posts can't be indexed automatically.

Can I contribute code?

Read CONTRIBUTING.md first. Open an issue before writing code — unsolicited PRs are closed without review.
