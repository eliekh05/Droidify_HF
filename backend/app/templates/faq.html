FAQ

Short answers. No padding.

The site

What is Droidify?

A live index of custom ROMs, recoveries, root tools, and guides for Android devices. Pulls from 20+ public sources in real time. No account needed.

Is it free?

Completely. No subscription, no paywall, no ads. MIT-licensed on GitHub.

Is Droidify affiliated with LineageOS, TWRP, or any ROM project?

No. Independent project. Reads their public APIs the same way a browser would.

Why is the first page load slow?

Cold start fetches data from 20+ sources simultaneously. Takes about two minutes. After that, everything is cached and fast.

How often does data refresh?

Most sources every two hours. Stock firmware every 24 hours.

Can I trust the download links?

Links go directly to the project’s own servers. We don’t proxy or modify files. Always verify the SHA256 checksum before flashing.

My device

Why does my device show zero ROMs?

Four possibilities: the ROM only exists as a forum post or Drive link (can’t be indexed), the cache hasn’t populated yet (wait two minutes), the device has no community ROM support, or the ROM uses a different codename. Open an issue with the codename and download URL if a ROM should be showing.

Why does a ROM show “Android unknown”?

Some sources don’t publish which Android version a build targets. We show “unknown” instead of guessing.

Where does the data come from?

LineageOS API and wiki, OrangeFox API, TWRP search.json, crDroid website, SourceForge file listings, GrapheneOS, postmarketOS GitLab, DivestOS, CalyxOS, /e/OS, Ubuntu Touch, NetHunter, customrombay.org, and more.

Flashing basics

What is a custom ROM?

A version of Android built by a community developer instead of your manufacturer. Replaces stock firmware entirely. Examples: LineageOS (clean), GrapheneOS (privacy), crDroid (feature-rich), /e/OS (no Google).

What is a custom recovery?

Replaces the stock recovery partition. Lets you flash ZIPs, create backups, and wipe partitions. Usually the first thing you flash after unlocking the bootloader. Common ones: TWRP, OrangeFox.

What does rooting mean?

Superuser access — like administrator on a PC. Lets you modify system files, remove bloatware, and run apps that need privileged access. Tools: Magisk (most compatible), KernelSU (kernel-level), APatch.

What does bootloader unlock mean?

The bootloader runs first on power-on and only allows manufacturer-signed software by default. Unlocking lets you flash anything. Wipes your device completely. On some devices it’s permanently blocked by the carrier.

Will flashing void my warranty?

On most devices yes. Samsung Knox trips permanently on unlock — no way back. Google Pixels can re-lock to restore security state. Check your device before proceeding.

Contributing

Can I request a missing ROM or device?

Open an issue with the codename and the ROM’s public download URL or API endpoint. Single-file Drive links or forum posts can’t be indexed automatically.

Can I contribute code?

Open an issue first before writing code. Unsolicited PRs are closed without review.
