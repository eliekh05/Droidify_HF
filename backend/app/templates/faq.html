FAQ

Short answers. No padding.

The site

What is Droidify?

A live index of custom ROMs, recoveries, root tools, and guides for Android devices. Pulls data from 20+ public sources in real time. No account needed to use it.

Is it free?

Completely. No subscription, no paywall, no ads. MIT-licensed and open source on GitHub.

Is Droidify affiliated with LineageOS, TWRP, or any ROM project?

No. Independent project. We read their public APIs the same way a browser would. No partnership, no endorsement from any of them.

Why is the first page load slow?

Cold start after a deploy fetches data from 20+ sources simultaneously. Usually takes about two minutes. After that everything is cached and loads fast.

How often does data refresh?

Most sources every two hours. Stock firmware every 24 hours.

Can I trust the download links?

Links go directly to the project’s own servers. We don’t proxy, modify, or re-host any files.

Install

How do I install Droidify on my phone?

Open the site in Chrome or Edge on Android. A banner will appear after a few seconds with an Install button. Tap it. Droidify lands on your home screen like any other app. Alternatively you can download the APK from the GitHub Releases page and sideload it.

How do I install it on iPhone or iPad?

Open the site in Safari. Tap the Share button at the bottom of the screen, then tap “Add to Home Screen”. Chrome and Firefox on iOS cannot install PWAs — it has to be Safari.

How do I install it on Windows?

Two options. The easier one: open the site in Chrome or Edge and use the install icon in the address bar. That installs it as a PWA with no warnings and no friction. The second option is the .msix package from the Releases page — see the Windows question below for the full story on that.

The Windows .msix shows a SmartScreen warning. Is it safe?

Yes it is safe. The source code is public on GitHub and you can read every line of it. The warning appears because the package is not signed through the Microsoft Store — it was generated by PWABuilder which signs it with its own certificate rather than a Microsoft-verified one. Microsoft treats anything not from their store with suspicion by default. Click “More info” then “Run anyway” to proceed. If you want to avoid the warning entirely, use Chrome or Edge to install from the browser — it works better and Microsoft cannot interfere with that.

Why does the Windows .msix have restrictions?

The .msix package was generated by PWABuilder, a Microsoft tool that wraps PWAs for the Windows Store. The package works fine for sideloading but Microsoft’s ecosystem is built around their store. Anything not distributed through it gets treated as a potential security risk regardless of what it actually is. The browser install (Chrome or Edge) is a proper first-class citizen on Windows and has none of these restrictions — that’s the better option.

How do I install it on Mac?

Open the site in Chrome or Edge and use the install icon in the address bar. In Safari, go to File → Add to Dock in the menu bar. The PWA then opens like any other app, separate from the browser.

What is the Android APK?

A TWA (Trusted Web Activity) — a thin native shell that opens the website full screen without a browser address bar. The APK itself contains almost no code. It is a verified pointer to the site. When the site updates, the app updates automatically with no new APK needed. You only need a new APK if the icon, package name, or signing key changes.

Do I need an account to use Droidify?

No. Everything works without signing in. Search, browse, read guides, check tools — all of it. Signing in with GitHub unlocks the watchlist and ROM alerts, nothing else.

My device

Why does my device show zero ROMs?

Four possibilities: the ROM only exists as a forum post or Google Drive link which cannot be indexed automatically, the cache hasn’t populated yet after a cold start, the device has no custom ROM support from any indexed source, or the ROM uses a different codename. Open an issue with the codename and download URL if a ROM should be showing.

Why does a ROM show “Android unknown”?

Some sources don’t include which Android version a build targets in their metadata. We show “unknown” rather than guess.

Where does the data come from?

LineageOS API and wiki, OrangeFox CDN, TWRP search.json, crDroid website, SourceForge file listings, GrapheneOS, postmarketOS GitLab, DivestOS, CalyxOS, /e/OS, Ubuntu Touch, NetHunter, customrombay.org, SamFW, and GitHub Releases.

Flashing basics

What is a custom ROM?

A version of Android built by a community developer instead of your manufacturer. Replaces stock firmware entirely. Examples: LineageOS (clean Android), GrapheneOS (privacy-first), crDroid (feature-rich), /e/OS (no Google).

What is a custom recovery?

Replaces the stock recovery partition. Lets you flash ZIPs, make backups, and wipe partitions. Usually the first thing you flash after unlocking the bootloader. Common ones: TWRP, OrangeFox.

What does rooting mean?

Superuser access — like administrator on Windows. Lets you modify system files, remove bloatware, and run apps that need privileged access. Tools: Magisk (most compatible), KernelSU (kernel-level), APatch.

What does bootloader unlock mean?

The bootloader runs first on power-on and only allows manufacturer-signed software by default. Unlocking it lets you flash anything. It completely wipes your device. On some devices and carriers it is permanently blocked.

Will flashing void my warranty?

On most devices yes. Samsung Knox trips permanently the moment you unlock — no way back. Google Pixels can re-lock the bootloader to restore the security state. Check your specific device before doing anything.

Contributing

Can I request a missing ROM or device?

Open an issue with the codename and the ROM’s public download URL or API endpoint. Single-file Drive links and forum posts cannot be indexed automatically.

Can I contribute code?

Open an issue before writing any code. Unsolicited pull requests are closed without review.
