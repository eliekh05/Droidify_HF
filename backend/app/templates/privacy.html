Privacy

Plain language. No legal padding. Last updated June 2026.

🚫 If you never sign in

We collect nothing. No cookies. No analytics. No fingerprinting. You browse, we serve pages. That’s the entire relationship.

🔓 If you sign in with GitHub

GitHub sends us three things and we store all three:

  • Your GitHub username — shown in the navbar.
  • Your avatar URL — shown next to your username. It’s a GitHub CDN link, not the image itself.
  • Your GitHub numeric ID — internal identifier so we know it’s you across sessions.

That’s it. No email, no followers, no repos, no private data. GitHub OAuth with read:user scope only reads public profile data.

📋 Your watchlist

Stored in our database tied to your GitHub ID. Up to 20 device codenames. Delete individual devices from the watchlist page. To delete your account entirely, open a GitHub issue.

🔔 ROM alerts

When new ROM builds are found for a device in your watchlist, an alert is created in our database and shown as a badge on the Watchlist nav item. No emails, no push notifications. In-app only.

🍪 Cookies

One cookie. A signed session token that keeps you logged in. Expires in 30 days. No tracking cookies. No third-party cookies.

🌎 Third parties
  • GitHub OAuth — for sign-in. Their privacy policy applies during that flow.
  • Bulma CSS from jsDelivr CDN — stylesheet only, no tracking.
  • HuggingFace — where the server runs. They see standard server logs (IP, timestamp, URL).
🕑 How long we keep data

Your account stays until you ask us to delete it. Open a GitHub issue to request deletion.

Short version: Sign in = we store your GitHub username, avatar URL, and numeric ID. Never sign in = we store nothing. No ads. No tracking. No selling data. Ever.

Questions? Open a GitHub issue.
