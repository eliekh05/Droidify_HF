# ⚖️ DMCA and Takedown Policy

## 🔍 What Droidify actually does

Droidify reads **publicly available metadata** — device names, codenames, ROM names, version numbers, and download links — from public APIs and web pages run by the original projects. That is it.

It does **not** host any files. It does not store, redistribute, mirror, copy, or touch any firmware, ROM, recovery image, or binary at any point.

Every download link on Droidify points directly to whoever made the file. _LineageOS mirrors. OrangeFox CDN. TWRP's own servers. GitHub Releases. SourceForge._ The files go from their servers to your device. **Droidify is not in that path anywhere.**

> This is what search engines do. This is what XDA does when it links to a download. Displaying public metadata and linking back to the original source is not redistribution and has never been redistribution.

---

## 🗑️ If you want your project removed

**No lawyer. No formal filing. No questions asked.**

Open a GitHub issue, tell us the project name and the URL on Droidify where it appears, and we will remove it **within 48 hours**.

If you want it **permanently blocked** so it never shows up again even if a scraper picks it up later, say so and we will add it to a blocklist. We have no interest in indexing anything from someone who does not want to be there.

---

## 📋 Formal DMCA notices

If you genuinely believe Droidify is hosting or redistributing a file you own the copyright to — _it is not_, but if you believe otherwise — open a GitHub issue and we will respond within 48 hours.

A DMCA notice needs to identify:
- The **specific** copyrighted work
- The **exact URL** on Droidify where you believe it appears

> A request that just says _"remove my project"_ is a standard takedown request — use that instead, it is simpler and faster.

---

## 💡 Why this project exists

Since Android 1.0 there has been **no single place** to find what is available for your device. All the data has been public the entire time. Nobody built the aggregator.

Droidify does not profit from anyone's work. **No ads. No fees.** Every link goes back to the original source. If your project appears here it is because it is publicly accessible and we are pointing people toward it — the same thing you did when you published it.
